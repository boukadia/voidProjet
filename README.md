# YouCanWin - Gamification Platform 🏆⚽

Bienvenue sur le projet **YouCanWin**, la plateforme digitale de référence qui aide les marques à engager leurs communautés autour du football via des mécaniques de gamification innovantes (cartes digitales à collectionner, quiz, tombola, etc.).

## 🚀 Fonctionnalités Principales

- **Activation instantanée** via des codes QR uniques.
- **Album digital & Collection** pour fidéliser les utilisateurs.
- **Récompenses exclusives** (tirages au sort, expériences VIP).
- **Catalogue de Jeux** : Pronostics, Quiz Football, Tombola.
- **Backoffice Analytics** pour un suivi en temps réel du ROI et de l'engagement.

## 💻 Stack Technologique

- **Frontend** : React.js, Vite
- **Styling** : Tailwind CSS v4 (Design system responsive et moderne)
- **Déploiement** : Vercel (Recommandé)

## 🛠️ Installation et Lancement

1. Clonez le projet :
```bash
git clone <votre-repo-url>
cd voidProjet
```

2. Installez les dépendances :
```bash
npm install
```

3. Lancez le serveur de développement :
```bash
npm run dev
```

La plateforme sera disponible sur `http://localhost:5173`.

## 📱 Design Complet & Responsive

L'interface de cette application a été développée en adoptant une approche responsive ("Mobile-first" par endroits, adaptable aux écrans larges et tablettes) :
- **Navigation (Navbar) dynamique** avec un menu "hamburger" sur smartphone.
- **Grilles auto-adaptatives** pour les sections (Jeux, Tarifs, Fonctionnalités).
- **Smooth scrolling** pour une navigation fluide et de qualité entre les différentes sections (`#offres`, `#jeux`, `#album`, `#apropos`).

## 🌐 Déploiement

Le site est conçu pour être déployé facilement sur Vercel. 
- Vous pouvez connecter directement votre dépôt GitHub à Vercel pour des déploiements automatiques.
- Ou utiliser le **Vercel CLI** : `npm i -g vercel` puis `vercel`.
